## File Responsibilities

- App.tsx - Main application orchestrator
- store/gpuStore.ts - Central state management for GPU data
- components/ConfigPanel/ConfigPanel.tsx - User controls for filtering and layout
- components/GraphCanvas/GraphCanvas.tsx - Cytoscape graph visualization
- models/componentGraph.ts - Transforms JSON data into graph nodes and edges
- models/componentNodeBuilder.ts - Creates standardized node objects
- models/componentNode.ts - Individual graph node structure
- types/component.ts - TypeScript interfaces for JSON structure
- components/Layout/AppLayout.tsx - UI layout wrapper

## Implementation Sequence

1. types/component.ts - Define TypeScript interfaces
2. models/componentNode.ts - Create node data structure
3. models/componentNodeBuilder.ts - Convert JSON to node objects
4. models/componentGraph.ts - Transform connections into graph edges
5. store/gpuStore.ts - Central data store implementation
6. components/GraphCanvas/GraphCanvas.tsx - Cytoscape visualization
7. App.tsx - Wire up main data flow
8. components/ConfigPanel/ConfigPanel.tsx - User controls
9. components/Layout/AppLayout.tsx - UI layout
